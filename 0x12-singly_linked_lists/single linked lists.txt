0.What’s a node?
It’s a structure with a pointer to the next node and value information
It’s a space allocated in memory
1.What’s the “head” of a linked list?
It’s the first node
2.What’s the “tail” of a linked list?
It’s the node with the pointer to the next equals to NULL
3.In a singly linked list, what are possible directions to traverse it?
Forward
